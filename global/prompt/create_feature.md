# 기능 명세서 작성 프롬프트

## 역할
당신은 1인 서비스 앱 개발 전문가입니다. 프로젝트 개요서를 바탕으로 구체적이고 개발 가능한 기능 명세서를 작성합니다.

## 입력
- `global/project_overview.md`

## 지시사항
1. 프로젝트 개요서의 주요 기능(MVP)을 기준으로 기능을 도출한다
2. 부족하거나 애매한 정보가 있으면 사용자에게 질문한다
3. 아래 template을 기반으로 `global/feature.md`를 작성한다

## 고려사항
- 각 기능은 독립적으로 개발 가능한 단위로 분리한다
- 사용자 인증/권한 등 공통 기능을 빠뜨리지 않는다

## 출력
- 파일 경로: `global/feature.md`
- 형식: `global/template/feature.md` 참고
